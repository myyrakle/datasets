# 한국 지하철 데이터셋 (2026-04-29)

한국 주요 도시의 지하철 노선 및 역 정보를 담은 데이터셋입니다.

---

## 파일 목록

| 파일명 | 설명 |
|---|---|
| `subway_lines.csv` | 지하철 노선 정보 |
| `subway_stations.csv` | 지하철 역 정보 |

---

## subway_lines.csv

지하철 노선별 기본 정보를 담고 있습니다.

| 컬럼명 | 타입 | 설명 |
|---|---|---|
| `id` | 정수 | 노선 고유 식별자 |
| `region` | 문자열 | 노선이 속한 도시/지역명 (예: 서울, 부산, 대구, 광주 등) |
| `line` | 문자열 | 노선명 (예: 1호선, 2호선, 대경선 등) |
| `station_count` | 정수 | 해당 노선의 역 수 |

---

## subway_stations.csv

지하철 역별 상세 정보를 담고 있습니다.

| 컬럼명 | 타입 | 설명 |
|---|---|---|
| `division` | 문자열 | 역이 속한 도시/지역명 (예: 서울, 부산, 대구, 광주 등) |
| `station_name` | 문자열 | 역의 대표 이름 |
| `station_names` | 문자열 배열 | 역의 모든 이름 목록 (병기역명 포함, PostgreSQL 배열 형식 `{...}`) |
| `latitude` | 실수 | 역의 위도 (WGS84 좌표계) |
| `longitude` | 실수 | 역의 경도 (WGS84 좌표계) |
| `lines` | 문자열 배열 | 해당 역을 경유하는 노선 목록 (환승역의 경우 복수 노선 포함, PostgreSQL 배열 형식 `{...}`) |
| `comment` | 문자열 | 비고 (선택적 추가 정보) |
